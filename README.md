# SMART SENSEI STYLE PROGRAM COMPILER

SMART SENSEI STYLE PROGRAM COMPILER is an intelligent coding assistant that:

- Reads handwritten or image-based code
- Detects programming language
- Executes code
- Explains errors
- Provides voice feedback

## Features

- OCR Code Reading
- Python / C++ / Java Detection
- Code Execution
- Error Explanation
- Voice Assistant
- GUI Software Interface

## Technologies Used

- Python
- CustomTkinter
- OCR
- Speech Recognition
- Text To Speech

## Run Project

```bash
pip install -r requirements.txt
python app.py
